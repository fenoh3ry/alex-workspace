## 1. Nature et enjeux du rapport

### Ce que c'est

Le rapport de stage n'est **pas** un simple témoignage — c'est un **travail de recherche** qui utilise l'expérience professionnelle comme **matériau empirique** pour répondre à une question scientifique.

### Trois objectifs causaux

```
Immersion professionnelle
    → Confrontation concepts théoriques / réalité terrain
    → Développement posture réflexive et critique
    → Préparation insertion professionnelle
```

| Ce n'est PAS | C'est |
|---|---|
| Chrono des tâches jour par jour | Analyse problématisée d'une expérience |
| Brochure promo de l'organisation | Posture de chercheur distancié |
| Témoignage subjectif | Travail scientifique avec contribution à la connaissance |
| État de l'art hors-sol | Articulation terrain ↔ théorie |
| Document sans précaution | Gestion rigoureuse de la confidentialité |

## 2. Chaîne de problématisation

### Le fil causal central

```
THÈME
  → CONSTAT (observation empirique)
    → TENSION (paradoxe / contradiction)
      → PROBLÉMATIQUE (question scientifique)
        → THÈSE (réponse défendue)
          → PLAN démonstratif
```

### Exemple concret

```
Thème      → Participation citoyenne en mairie
Constat    → Dispositifs de consultation mis en place
Tension    → Espaces de démocratie délibérative OU instruments de légitimation ?
Problématique → Comment concilier ouverture démocratique + contraintes bureaucratiques ?
```

### Règle d'or : le titre doit être problématisé

❌ *« Stage de 6 mois au service communication de La Voix du Nord »*

✅ *« L'IA générative dans les rédactions : menace ou opportunité pour les journalistes ? Retour sur six mois à La Voix du Nord »*

**Cause** : un titre non problématisé → rapport sans angle analytique → échec évaluatif.

## 3. Préparer avant le stage

### La clé : choisir l'angle AVANT le stage

```
Angle identifié tôt
  → Observations ciblées durant le stage
    → Matériau empirique riche et pertinent
      → Analyse fluide et naturelle

vs.

Angle cherché a posteriori
  → Analyse forcée, artificielle
    → Terrain sous-exploité
```

### Trois sources d'angle

- **Mission particulière** → ex : lobbying → mobilisation « société civile » / « policy networks »
- **Observation transversale** → management, communication politique, logiques de financement…
- **Tension observée** → écart doctrine/pratique, contradiction objectifs/moyens

### Journal de terrain — règles

| Règle | Pourquoi |
|---|---|
| Régularité (quotidien/hebdo) | Patterns invisibles au jour le jour |
| Séparer faits / interprétations | Rigueur analytique |
| Anonymiser immédiatement | Éviter les fuites |
| Dater chaque entrée | Relecture chronologique utile |

## 4. Structure du rapport

### Architecture causale

```
Page de couverture (titre problématisé)
  → Résumé / Abstract (1p max)
  → Remerciements
  → Acronymes + Table des illustrations + Sommaire
  → INTRODUCTION (2-3p)
      Contexte → Définitions → Problématisation → Thèse + Plan annoncé
  → PARTIE I (15-20p) — Présentation critique de l'organisation
      Macro → Micro / Missions analytiques / Bilan compétences
  → PARTIE II (20-30p) — Analyse problématisée ⚠️ CŒUR
      Terrain ↔ Théorie / Argumentation / Sources empiriques
  → CONCLUSION (1-2p)
      Récapitulation → Bilan critique → Perspectives
  → BIBLIOGRAPHIE (1-2p, norme homogène)
  → TABLE DES MATIÈRES
  → ANNEXES (entretien-métier + documents)
```

### Ordre de rédaction

```
Partie I (échauffement, structure le matériau)
  → Partie II (cœur analytique)
    → Conclusion
      → Bibliographie
        → INTRODUCTION EN DERNIER ← crucial
          (on ne peut annoncer que ce qu'on a démontré)
```

### Équilibre des volumes

| Section | Part du rapport | Danger si déséquilibré |
|---|---|---|
| Introduction | ≤ 10% | Problématique floue |
| Partie I | ~30-35% | > 40% = difficulté à analyser |
| Partie II | ~40-50% | < 30% = théorisation insuffisante |
| Conclusion + biblio + annexes | ~15% | — |

## 5. Articulation terrain ↔ théorie

### Deux mouvements causaux

**↗️ Terrain → Théorie (ascendant)**

```
Observation précise
  → Identification d'un concept (nommer ce qu'on a vu)
    → Généralisation au-delà du cas particulier
```

*Exemple* : agents passent 60% du temps sur rapports → Lipsky, *street-level bureaucracy* → la paperasserie = mécanisme d'ajustement entre objectifs contradictoires.

**↘️ Théorie → Terrain (descendant)**

```
Concept théorique
  → Interrogation du terrain (la théorie s'applique-t-elle ?)
    → Nuance ou critique (écarts, limites)
```

*Exemple* : bureaucratie weberienne → règles suivies ? hiérarchie effective ? → coexistence bureaucratisation/débureaucratisation (Crozier & Friedberg).

### Trois fonctions de la théorie

```
NOMMER   → Donner un vocabulaire aux observations
EXPLIQUER → Produire du sens causal
GÉNÉRALISER → Dépasser le cas singulier
```

### Critères de sélection d'une référence

- Le concept **nomme-t-il** ce qu'on a observé ?
- L'auteur permet-il de **généraliser** au-delà du cas ?
- La référence est-elle en **dialogue** avec d'autres travaux ?

### Règle impérative

> **Chaque paragraphe théorique doit être illustré par une observation de terrain.** Posez-vous : *« Qu'est-ce que j'ai vu qui confirme, infirme ou nuance cette théorie ? »* Si pas de réponse → paragraphe superflu.

## 6. Règles d'écriture académique

### Posture : le chercheur, pas le salarié

```
Posture salarié    → « Notre service fait un travail remarquable. »
Posture chercheur  → « Les agents mobilisent un discours de la "passion"
                     pour compenser l'absence de reconnaissance matérielle
                     → Boltanski & Chiapello, nouvel esprit du capitalisme. »
```

### Temporalité

- **Passé** → faits du stage
- **Présent** → analyse et références théoriques

### 1re personne : mesurée

✅ *« J'ai pu observer »*, *« Mon expérience m'a permis de constater »*
❌ *« Je pense que »*, *« À mon avis »* → préférez *« Cette observation suggère »*

### Citation : toujours intégrée

```
❌ Citation orpheline : on cite sans expliquer pourquoi
✅ Citation intégrée : introduction → citation → commentaire
```

### Confidentialité — trois niveaux

| Niveau | Action |
|---|---|
| **Libre** | Sources transparentes, nom de l'organisation cité |
| **Restreinte** | Anonymisation (pseudonyme, fonctions, ordres de grandeur) |
| **Interdite** | « CONFIDENTIEL » en rouge sur couverture + motifs |

## 7. Trois syndromes à éviter

### Syndrome 1 : Le compte-rendu de tâches

```
Cause     → Rédaction chronologique jour par jour
Symptôme  → « Lundi j'ai rédigé. Mardi j'ai classé. »
Conséquence → Pas de problématique, pas de théorie = échec académique
Remède    → Chaque mission requalifiée en termes analytiques
```

### Syndrome 2 : La brochure promotionnelle

```
Cause     → Posture de salarié satisfait, pas de distance
Symptôme  → Organisation toujours présentée favorablement
Conséquence → Discours non interrogé = pas d'analyse
Remède    → Interroger systématiquement TOUS les discours, y compris les séduisants
```

### Syndrome 3 : La théorie sans terrain

```
Cause     → Exposé de cours plaqué sur le stage
Symptôme  → Stage invisible dans la Partie II
Conséquence → N'importe quel stage ferait l'affaire = travail non fait
Remède    → Chaque § théorique = 1 observation terrain. Sinon → supprimer.
```

## 8. Évaluation et dépôt

### Ce que valorise le jury

```
Problématisation claire et pertinente
  → Articulation terrain/théorie maîtrisée
    → Posture réflexive et distanciée
      → Bibliographie pertinente (qualité > quantité)
        → Soin formel (orthographe, mise en page, normes)
```

### Ce qui pénalise

- Absence de problématique → compte-rendu descriptif
- Posture promotionnelle → pas de distance critique
- Texte sans références → pas de travail théorique
- Plagiat → échec immédiat
- Manque de soin formel → crédibilité affaiblie

### Normes bibliographiques

Choisir **UNE** norme et s'y tenir : **ISO 690** (sciences sociales françaises) ou **APA 7e** (anglais).

### Annexe obligatoire : entretien-métier

```
Guide d'entretien préparé en amont
  → Compte-rendu (2 pages)
    → Analyse critique : pourquoi cet interlocuteur ? Quoi appris ?
```

## Récapitulatif causal final

```
Angle identifié AVANT le stage
  → Journal de terrain régulier
    → Matériau empirique structuré
      → Problématisation (tension → question)
        → Argumentation (terrain ↔ théorie, va-et-vient)
          → Posture de chercheur distancié
            → Rédaction Introduction EN DERNIER
              → Relecture par un tiers
                → Dépôt dans les délais
```
