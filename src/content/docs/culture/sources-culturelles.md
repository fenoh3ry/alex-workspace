# Sources culturelles

Cette section rassemble les ressources culturelles liées au domaine de recherche.

> **Note** : Le dossier source (`/culture/`) contient exclusivement des fichiers au format PDF qui n'ont pas pu être directement convertis en contenu Markdown pour ce site. Les documents sont accessibles dans le dossier d'origine :

- `culture/pdfs/approche culturelle.pdf`

Une conversion manuelle ou par OCR de ces documents PDF en Markdown permettra d'enrichir cette section avec le contenu détaillé.
