We're excited to announce the launch of the **Command Palette**, a fast, keyboard-driven way to navigate your workspace, trigger actions, and streamline your development flow.

## Overview

The Command Palette allows you to access nearly every feature of the platform without leaving your keyboard. Whether you're switching views, toggling a setting, or running a command — it's all just a few keystrokes away.

## Key Features

### Global Search and Navigation  
Quickly move between sections of your dashboard, projects, or documentation without using the mouse.

### Command Execution  
Run core actions like deploying a project, clearing cache, or toggling themes with instant feedback.

### Context Awareness  
The palette dynamically adapts based on your location in the app. For example:
- In a project view, it prioritizes project-specific commands.
- On settings pages, it suggests relevant configuration options.

### Keyboard Shortcut  
Open the Command Palette from anywhere using `Cmd + K` (or `Ctrl + K` on Windows/Linux).

## Designed for Speed

This update focuses on removing friction — no more digging through menus or remembering where settings live. Just type, select, and go.

## What’s Next

We’re already working on palette extensions so you can add your own custom commands, shortcuts, and workflow integrations. Expect updates soon.
